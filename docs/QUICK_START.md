# ⚡ 快速上手指南

只需 3 步，即可在 VR 设备中使用双目立体视觉！

## 📋 准备工作

确保你有：
- ✅ 一台带有 USB 双目相机的电脑
- ✅ 一台 VR 设备（Quest 3、PICO 4 等）
- ✅ 电脑和 VR 设备在同一个 WiFi 网络

## 🚀 三步启动

### 第 1 步：安装依赖

打开终端/命令行，运行：

```bash
pip install opencv-python websockets numpy
```

### 第 2 步：启动服务器

```bash
python start.py
```

你会看到类似这样的输出：

```
======================================================================
🚀 立体视觉服务器启动中...
======================================================================

======================================================================
📱 其他设备访问方法:
======================================================================
1. 在VR设备或手机浏览器打开:
   https://192.168.1.100:8445

2. 浏览器会提示证书不安全，选择'继续前往'或'信任此证书'

3. 进入导航页面，选择'进入VR模式'或'打开2D查看器'
======================================================================
```

**记下你看到的 IP 地址！**（上面的例子是 `192.168.1.100`）

### 第 3 步：在 VR 设备中访问

1. **打开 VR 头显的浏览器**
   - Quest: 在应用库中找到"浏览器"
   - PICO: 在应用库中找到"浏览器"或"网页浏览器"

2. **输入IP地址**
   ```
   https://你的IP:8445
   ```
   例如：`https://192.168.1.100:8445`

3. **处理证书警告**
   - 浏览器会显示"不安全"或"证书无效"
   - 这是正常的！因为我们使用自签名证书
   - 点击"高级" → "继续访问"或"仍然访问"

4. **选择模式**
   - 点击 **"进入VR模式"** 蓝色按钮 🥽

5. **开始体验**
   - 页面加载后自动连接视频流
   - 点击 **"进入 VR"** 按钮即可

## 🎯 测试模式

如果你没有真实的双目相机，服务器会自动进入**测试模式**，生成虚拟画面供你预览和测试系统功能。

## ❓ 遇到问题？

### 问题 1: 找不到 IP 地址

**Windows**:
```bash
ipconfig
```
查找 "IPv4 地址" 或 "无线局域网适配器"

**Mac/Linux**:
```bash
ifconfig
```
查找 "inet" 后面的地址（通常是 192.168.x.x）

### 问题 2: VR 设备无法访问

确认：
- ✅ 电脑和 VR 设备在同一个 WiFi
- ✅ 防火墙允许 8445 和 8765 端口
- ✅ 使用 HTTPS（不是 HTTP）

### 问题 3: 页面加载但没有视频

1. 检查服务器是否正常运行
2. 刷新页面
3. 查看浏览器控制台是否有错误
